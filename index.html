About

A pure-Python pipeline for extracting entities (people, organizations, locations, dates, tech) and relations (founded_by, located_in, works_for, …) from unstructured text, building a knowledge graph, then running graph algorithms over it.

Entirely pattern + rule based — no spaCy, no transformers. Built to understand how the dependencies-free path can take you further than you'd expect.
